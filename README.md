# Project Overview

This project is built using a modern web stack composed of a backend API, a frontend framework, and cloud services.

## Technologies

- **Flask API** – Backend service that provides the REST API.
- **Angular** – Frontend framework used to build the user interface.
- **Firebase** – Cloud platform used for backend services and real-time features.

## Architecture

The application follows a simple architecture:

Frontend (Angular) → Backend API (Flask) → Cloud Services (Firebase)
